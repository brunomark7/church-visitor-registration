Sistema web para cadastro de visitantes em igrejas, desenvolvido com foco em simplicidade, responsividade e experiência do usuário.


---

## 📌 Sobre o Projeto

Este sistema foi desenvolvido para facilitar o cadastro de visitantes durante cultos e eventos de igreja.

O objetivo é oferecer uma experiência rápida e intuitiva via dispositivos móveis, permitindo que o visitante realize seu cadastro em poucos segundos.

---

## ⚙️ Funcionalidades

- Cadastro de visitantes
- Popup/modal interativo
- Validação de campos (nome e telefone)
- Mensagens de erro personalizadas
- Página de boas-vindas
- Personalização da mensagem com o nome do visitante
- Uso de `sessionStorage` para troca de dados entre páginas
- Layout responsivo para dispositivos móveis
