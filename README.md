# Church Visitor Registration

Sistema web para cadastro de visitantes em igrejas.

## Sobre o Projeto

Este projeto foi desenvolvido para facilitar o cadastro de visitantes durante cultos e eventos da igreja.

A proposta é permitir que o visitante realize seu cadastro de forma rápida e prática por meio de um formulário acessado pelo celular.

## Funcionalidades

* Cadastro de visitantes
* Interface amigável
* Coleta de informações de contato
* Layout responsivo para dispositivos móveis

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Git
* GitHub
